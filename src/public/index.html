Notes4All

This is a note taking API made by @Adrian-rospx

It uses ExpressJS and SQLite3 for data persistency.

API usage

Authentication

Register from /auth/register with a POST request

curl -X POST https://rospxrpi.go.ro/notes/auth/register \
    -H "Content-Type: application/json" \
    -d "{'username':'<USERNAME>', 'password':'<PASSWORD>'}"

Login from /auth/login with a POST request

curl -X POST https://rospxrpi.go.ro/notes/auth/login \
    -H "Content-Type: application/json" \
    -d "{'username':'<USERNAME>', 'password':'<PASSWORD>'}"

Use the JSON Web Token in the Authorization header for API calls
