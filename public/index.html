Notes4All

This is a note taking API made by @Adrian-rospx

It uses ExpressJS and SQLite3 for data persistency.

https://github.com/Adrian-rospx/notes4all

API usage

Authentication

Register from /auth/register with a POST request

curl -X POST https://notes4all.com/auth/register \
    -H "Content-Type: application/json" \
    -d "{'username':'<USERNAME>', 'password':'<PASSWORD>'}"

Login from /auth/login with a POST request

curl -X POST https://notes4all.com/auth/login \
    -H "Content-Type: application/json" \
    -d "{'username':'<USERNAME>', 'password':'<PASSWORD>'}"

Use the JSON Web Token in the Authorization header for API calls 

    -H "Authorization: <TOKEN>"

Notes API

Retrieve all of your notes with a GET request from /api 

curl https://notes4all.com/api

Retrieve a specific note by specifying its /:id

curl https://notes4all.com/api/<ID>

Create a note by passing a JSON of title and content 

curl -X POST https://notes4all.com/api \
    -H "Content-Type: application/json" \
    -d "{'title':'<TITLE>', 'content':'<CONTENT>'}"

Modify a note's content with a PATCH request using its id 

curl -X PATCH https://notes4all.com/api/<ID> \
    -H "Content-Type: application/json" \
    -d "{'content':'<NEW_CONTENT>'}"

Delete a note with DELETE request using its id 

curl -X DELETE https://notes4all.com/api/<ID>
